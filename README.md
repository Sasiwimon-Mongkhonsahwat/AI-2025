**ระบบช่วยตัดสินใจเลือกสายอาชีพด้วยปัญญาประดิษฐ์**
โครงงานนี้มีที่มาจากปัญหาของนักศึกษาที่ขาดข้อมูลในการตัดสินใจเลือกอาชีพให้เหมาะสมกับความถนัดและทักษะของตนเอง จึงมีแนวคิดที่จะใช้ AI เข้ามาช่วยวิเคราะห์ข้อมูลส่วนบุคคล เพื่อเพิ่มประสิทธิภาพในการวางแผนอนาคตและลดความสับสนในการเลือกสายอาชีพ

**🚀 ฟีเจอร์หลัก (Features)**
  - การวิเคราะห์ข้อมูลบุคคลด้วย AI: ระบบจะนำข้อมูลส่วนตัวของนักศึกษามาประมวลผลผ่านโมเดลปัญญาประดิษฐ์เพื่อหาความสอดคล้องกับสายอาชีพ
  - ระบบประเมินความพร้อมและศักยภาพ: วิเคราะห์ข้อมูลรอบด้าน ประกอบด้วย:
        - แบบสอบถามวัดความสนใจ (Interest Survey)
        - การประเมินทักษะที่ถนัด (Skill Assessment)
  - การจัดอันดับสายอาชีพที่เหมาะสม: แสดงผลลัพธ์เป็นรายการสายอาชีพที่แนะนำจำนวน 3–5 อันดับ เพื่อให้ผู้ใช้งานมีทางเลือกที่หลากหลายแต่ยังคงความแม่นยำ

**👥 สมาชิกและภาระงาน (Team Members & Responsibilities)**
ตารางแสดงการแบ่งงานรายสัปดาห์:

|สัปดาห์ที่(Week)|ศศิวิมล มงคลสวัสดิ์ |
| :--- | :--- |
| **Week 1** | วางแผนโครงงาน |
| **Week 2** | ออกแบบโครงงาน |
